# Handoffs

This directory stores file-backed Markdown handoffs and reports between Codex and Claude Code.

Use:

- `YYYY-MM-DD-<topic>-codex-to-claude.md` for Codex execution briefs.
- `YYYY-MM-DD-<topic>-claude-report.md` for Claude Code execution reports.

Follow the templates in `docs/agent-memory/agent-communication.md`.

Do not store secrets, full Cookie values, `.env` contents, npm tokens, GitHub tokens, or private credentials here.
