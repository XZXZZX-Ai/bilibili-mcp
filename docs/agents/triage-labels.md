# Triage Labels

The Matt Pocock skills use five canonical triage roles. This repository keeps the default label strings.

| Matt role | GitHub label | Meaning |
|---|---|---|
| `needs-triage` | `needs-triage` | Maintainer evaluation is required |
| `needs-info` | `needs-info` | Waiting for more information from the reporter |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for a bounded agent task |
| `ready-for-human` | `ready-for-human` | Requires human implementation or judgment |
| `wontfix` | `wontfix` | Will not be actioned |

When a Matt skill names a canonical role, use the corresponding GitHub label from this table.
